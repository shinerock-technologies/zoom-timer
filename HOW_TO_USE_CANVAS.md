## Quick Start Guide

### 1. Start the App

```bash
npm start
```

### 2. Join a Zoom Meeting

Create or join any Zoom meeting

### 3. Open the Timer App

Load the Multi Timer app in the meeting

### 4. Enable Canvas Mode

Click the **Canvas** button in the top-right header (it will turn blue)

### 5. Select Timer Overlay as Camera

1. In Zoom, click the **^** arrow next to your video
2. Select **"Timer Overlay"** from the camera list
3. Your video will now show the timer overlay

### 6. Start a Timer

Create and start any timer - it will appear in your video feed!

## Troubleshooting

**Nothing happening?**

1. Check that Canvas button is blue (enabled)
2. Verify you selected "Timer Overlay" as camera
3. Make sure a timer is running
4. Check Zoom Client version (5.10.6+)

**Can't find "Timer Overlay" in camera list?**

1. Enable Canvas mode first (click button)
2. Wait a few seconds
3. Refresh camera list
4. Check browser console for errors

**Timer not updating?**

1. Disable and re-enable Canvas
2. Restart the timer
3. Check that timer is actually running
